# 📺 Mr.Rm19 TV Streaming Indonesia

Selamat datang di repositori **Mr.Rm19 TV Streaming Indonesia**! Proyek ini adalah platform pemutar siaran televisi (TV) Indonesia berbasis web statis yang sangat ringan, responsif, dan gratis. Dikembangkan sepenuhnya menggunakan HTML dan CSS modern, proyek ini memanfaatkan metode *iFrame Embed* dan *Live Stream API* resmi untuk menyajikan tayangan berkualitas tinggi tanpa membebani server lokal.

 Live Website: [https://Rm19x.github.io/TV-indonesia/](https://Rm19x.github.io/TV-indonesia/)
---

##  Fitur Utama

*   **Antarmuka Gelap Modern (Dark Mode):** Desain UI yang nyaman di mata, terinspirasi dari skema warna GitHub.
*   **Kategori Channel yang Rapi:** Saluran TV dikelompokkan berdasarkan kategori (*Nasional, Berita/Bisnis, TVRI Jaringan, dan Daerah*).
*   **Metode iFrame & YouTube Live Player:** Menggunakan jalur resmi yang stabil, aman, dan meminimalkan tautan mati (*broken links*).
*   **Desain Responsif:** Tampilan otomatis menyesuaikan layar, sangat nyaman diakses baik melalui Laptop/PC maupun Smartphone (HP).
*   **Tanpa Backend / Serverless:** Berjalan sepenuhnya di sisi klien (client-side) lewat layanan gratis GitHub Pages.
